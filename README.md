# 🐦 Flappy Bird Clone

A classic Flappy Bird clone built with Python and Pygame! Flap your wings to navigate through pipes, avoid crashing, and beat your high score.

## 🎮 Features
- **Classic Gameplay:** Flap to fly and dodge randomly generated pipes.
- **Smooth Animations:** The bird flaps its wings and rotates dynamically based on its velocity.
- **Pixel-Perfect Collisions:** Accurate hit detection using Pygame masks.
- **Sound Effects:** Includes audio feedback for flapping, scoring, and game over.
- **Score Tracking:** Keep track of your current score and your all-time best run.
- **Seamless Scrolling:** Smoothly scrolling background and ground.

## 🕹️ Controls
- **Flap / Start / Restart:** Press `SPACEBAR` or `LEFT CLICK`.

## ⚙️ Requirements
- Python 3.x
- Pygame
